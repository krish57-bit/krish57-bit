### 🥛 PARAKH — Non-Invasive Milk Quality Analysis System
> **Capstone Project · IoT + AI + Embedded Systems · Ongoing**

A real-world IoT+ML pipeline for detecting milk adulteration (water dilution + urea) without opening the container — using RF backscatter sensing, embedded ML, and a mobile app.

**Hardware:** Coplanar capacitor sensing (copper foil tape, 2mm gap on PET bottle) · TI RF430FRL152H NFC IC (14-bit Σ-Δ ADC) · ESP32 + Adafruit PN532 gateway · Keysight E5063A VNA (ground-truth data collection)

**Software:** Python VNA automation scripts · Edge Impulse TinyML pipeline · React Native mobile app · Spring Boot + PostgreSQL backend · Alien 9640 "Squiggle" UHF wet inlay tags

**ML Pipeline:** S11 feature extraction (magnitude + phase) → 8-feature flat matrix → Decision Tree classifier → Edge deployment via Edge Impulse

```
Tech: Python · ESP32 · NFC/RFID · Spring Boot · React Native · PostgreSQL · Edge Impulse · Scikit-learn
```
